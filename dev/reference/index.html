LaplaceRedux.fit! — Method
fit!(la::Laplace,data)

Fits the Laplace approximation for a data set.

Examples

using Flux, LaplaceRedux
x, y = LaplaceRedux.Data.toy_data_linear()
data = zip(x,y)
nn = Chain(Dense(2,1))
la = Laplace(nn)
fit!(la, data)
